# راهنمای اجرای سایت در لوکال هاست

## روش 1: استفاده از فایل Batch (ساده‌ترین روش)

1. روی فایل `start-server.bat` دوبار کلیک کنید
2. مرورگر را باز کنید و به آدرس زیر بروید:
   ```
   http://localhost:8000
   ```
3. برای توقف سرور، پنجره Command Prompt را ببندید یا Ctrl+C را فشار دهید

## روش 2: استفاده از Command Prompt

1. Command Prompt را باز کنید
2. به پوشه پروژه بروید:
   ```cmd
   cd "C:\Users\Raven\Desktop\دانشگاه\صیفوری\Medilab"
   ```
3. دستور زیر را اجرا کنید:
   ```cmd
   python -m http.server 8000
   ```
4. مرورگر را باز کنید و به آدرس `http://localhost:8000` بروید

## روش 3: استفاده از PowerShell

1. PowerShell را باز کنید
2. به پوشه پروژه بروید
3. دستور زیر را اجرا کنید:
   ```powershell
   python -m http.server 8000
   ```

## نکات مهم:

- پورت پیش‌فرض: **8000**
- اگر پورت 8000 اشغال باشد، می‌توانید پورت دیگری استفاده کنید:
  ```cmd
  python -m http.server 8080
  ```
- برای دسترسی از سایر دستگاه‌ها در شبکه محلی:
  ```
  http://[آدرس IP شما]:8000
  ```

## ساختار پروژه:

- `index.html` - صفحه اصلی سایت
- `assets/` - فایل‌های CSS، JavaScript و تصاویر
- `forms/` - فایل‌های PHP برای فرم‌ها
