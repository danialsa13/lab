# سیستم مدیریت آزمایشگاه آزمایاب

یک سیستم کامل برای مدیریت آزمایشگاه با دو بخش اصلی: **بخش ادمین** و **بخش بیمار**.

## ویژگی‌ها

### بخش ادمین آزمایشگاه:
- ✅ ورود ادمین با نام کاربری و رمز عبور
- ✅ افزودن بیمار جدید (شماره تماس و نام)
- ✅ افزودن آزمایش: انتخاب بیمار و آپلود برگه آزمایش
- ✅ مشاهده لیست تمام بیماران
- ✅ مشاهده لیست تمام آزمایش‌ها

### بخش بیمار:
- ✅ ورود با شماره تماس و کد تایید
- ✅ مشاهده لیست آزمایش‌های خود
- ✅ مشاهده و دانلود نتایج آزمایش‌ها

### سایر ویژگی‌ها:
- ✅ رابط کاربری فارسی و RTL
- ✅ طراحی واکنش‌گرا (Responsive)
- ✅ استفاده از localStorage برای ذخیره داده‌ها
- ✅ صفحه درباره ما
- ✅ رابط کاربری مدرن و زیبا

## نحوه استفاده

### راه‌اندازی سرور محلی:

1. **روش ساده (Windows):**
   - روی فایل `start-server.bat` دوبار کلیک کنید

2. **روش دستی:**
   ```bash
   python -m http.server 8000
   ```

3. باز کردن در مرورگر:
   ```
   http://localhost:8000
   ```

## اطلاعات ورود

### ادمین:
- **نام کاربری:** `danix`
- **رمز عبور:** `danix`

### بیمار:
- برای ورود بیمار، ابتدا باید توسط ادمین در سیستم ثبت شود
- سپس با شماره تماس ثبت شده و کد تایید می‌تواند وارد شود
